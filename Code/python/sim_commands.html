Parameter reference — run_simulation.py (local)
ParameterTypeDefaultDescription
--plansstringall"all" or comma-separated plan IDs (e.g. AZ06,CA10). MA50 always excluded.
--stagestringbothdetal · asset · both. "detal" runs deterministic A/L; "asset" runs stochastic simulation (requires existing detAL pkl); "both" chains them sequentially.
--parallelint4Max concurrent plan processes. Acts as a ceiling — a new plan starts as soon as a slot frees up, not in rigid batches.
--run-tagstring062026Output folder label. Results go to Results/Runs/[run-tag]/[PLAN]/.
--plan-yearint2022Model data year passed to the A/L script via --plan-year.
--num-simint1000Number of asset-return simulations per plan (asset stage only).
--seedint(none)Common market shock seed, shared by ALL plans (asset stage): simulation column n is the same market history for every plan. If omitted, one seed is generated and printed.
--tier-filestringplanchanges_main_2022_clean.xlsxTier parameter file name inside Common_Data/.
--plan-filestring(none)Explicit path to a plan list file. Defaults to Pipeline/[run-tag]/plans_38.txt.
--fastswitchoffUse fast/Main_PensionModel.py (optimized loops; ~10x faster than the standard version after the 2026-06-10 core.py optimization). Results are bit-identical to the standard version.
--workersintcpu_countThread-pool workers for PVNC (fast mode only). Profiling (2026-06-10) showed PVNC threads are GIL-serialized, so extra workers add overhead without speedup: use --workers 1 and put all parallelism in --parallel instead. Worker count does not affect results.
--skip-existing-detalswitchoffSkip plans that already have a detAL pkl. Use to rerun only failed plans.
--skip-existing-assetswitchoffSkip plans that already have an asset pkl.
--overwriteswitchoffForce overwrite existing outputs.
--dry-runswitchoffPrint commands without executing them.
--date-runstring(none)Optional date_run label forwarded to the A/L script.
--discount-overridefloat(none)Forwarded to the fast detal runner: replaces the plan GASB discount rate in AAL/PVNC (e.g. an AAA yield). Scenario lever; see launcher.ipynb. asset_simulation.py has further scenario levers (--contrib-add, --policy-start, --contrib-always, --equity-share, --derisk-to, --derisk-years, --stock-premium, --stock-vol, --detal-run-tag).
